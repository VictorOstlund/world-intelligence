## Goal
Scaffold the A2 standalone Next.js 15 project with TypeScript, initialize SQLite schema (reports, config, users tables), and implement the auth system (bcrypt password hashing, JWT via jose, httpOnly cookie sessions). This is the foundation everything else builds on — no LLM calls, no RSS fetching, no pipeline yet.

**`lib/schema.sql`** (inline in `lib/db.ts` as string, executed via `db.exec()`):
```sql
CREATE TABLE IF NOT EXISTS reports (
  id TEXT PRIMARY KEY,
  created_at INTEGER NOT NULL,
  schedule TEXT NOT NULL,
  categories TEXT NOT NULL,  -- JSON array
  summary TEXT NOT NULL,
  body TEXT NOT NULL,
  cost_usd REAL NOT NULL DEFAULT 0,
  triage_model TEXT NOT NULL,
  synthesis_model TEXT NOT NULL,
  item_count INTEGER NOT NULL DEFAULT 0,
  source_count INTEGER NOT NULL DEFAULT 0
);

CREATE TABLE IF NOT EXISTS config (
  id INTEGER PRIMARY KEY CHECK (id = 1),  -- single row enforced
  active_provider TEXT NOT NULL DEFAULT 'anthropic',
  triage_model TEXT NOT NULL DEFAULT 'gemini-1.5-flash-8b',
  synthesis_model TEXT NOT NULL DEFAULT 'claude-sonnet-4-6',
  triage_fallbacks TEXT NOT NULL DEFAULT '[]',   -- JSON array, max 2
  synthesis_fallbacks TEXT NOT NULL DEFAULT '[]', -- JSON array, max 2
  schedule_hours INTEGER NOT NULL DEFAULT 6,
  category_config TEXT NOT NULL DEFAULT '{}',    -- JSON object: {[cat]: {enabled, itemBudget}}
  providers TEXT NOT NULL DEFAULT '{}'           -- JSON: never returned to client
);

CREATE TABLE IF NOT EXISTS users (
  id TEXT PRIMARY KEY,
  username TEXT UNIQUE NOT NULL,
  password_hash TEXT NOT NULL,
  created_at INTEGER NOT NULL
);

CREATE VIRTUAL TABLE IF NOT EXISTS reports_fts USING fts5(
  id UNINDEXED,
  body,
  summary,
  content='reports',
  content_rowid='rowid'
);
```

**`lib/auth.ts`**:
- `hashPassword(plain: string): Promise<string>` — bcryptjs, 12 rounds
- `verifyPassword(plain: string, hash: string): Promise<boolean>`
- `signJwt(payload: { userId: string, username: string }): Promise<string>` — uses `process.env.JWT_SECRET`, 7d expiry
- `verifyJwt(token: string): Promise<{ userId: string, username: string } | null>`
- `getSession(req: Request): Promise<{ userId: string, username: string } | null>` — reads `session` httpOnly cookie

**`app/api/auth/login/route.ts`**:
- POST — accepts `{ username, password }`
- Looks up user in SQLite by username
- Verifies bcrypt hash
- On success: signs JWT, sets httpOnly cookie `session`, returns `{ ok: true }`
- On failure: returns 401 `{ error: 'invalid credentials' }`
- Never returns password hash or any credential

**`app/api/auth/logout/route.ts`**:
- POST — clears `session` cookie, returns 200

**`app/api/auth/me/route.ts`**:
- GET — returns `{ username }` if session valid, 401 if not

**`middleware.ts`** (Next.js edge middleware):
- Protects all routes except `/login` and `/api/auth/*`
- Reads `session` cookie, verifies JWT
- Redirects to `/login` if invalid
